# playwright-ipl-framework

## Tech Stack
- Playwright
- Java
- Maven
- TestNG

## Framework Features
- Parallel execution
- Auto waits
- Reusable BasePage methods
- Page Object Model

## Project Structure

src/test/java
 ├── base
 ├── pages
 ├── tests
 └── utils

## How to Run Tests

Run individual tests from IntelliJ
OR
Run all tests using:
mvn test
OR
Run testng.xml
